# Changelog

## 1.0.0 (Initial Release)

- Initial release of the Bluetooth Low Energy plugin
- Support for scanning BLE devices
- Support for connecting/disconnecting to devices
- Support for discovering services and characteristics
- Support for reading/writing characteristic values
- Support for notifications and indications
- Support for reading/writing descriptor values
- Support for reading RSSI
- Support for MTU negotiation (Android)
- Support for connection priority (Android)
- Support for BLE advertising (peripheral mode)
- Web Bluetooth API support for web platform
