# Contributing

We welcome contributions to this project!

## Development Setup

1. Clone the repository
2. Install dependencies with `npm install`
3. Build the plugin with `npm run build`

## Building

```bash
npm run build
```

## Testing

### Web
```bash
npm run verify:web
```

### iOS
```bash
npm run verify:ios
```

### Android
```bash
npm run verify:android
```

## Formatting

```bash
npm run fmt
```

## Submitting Changes

1. Create a new branch for your changes
2. Make your changes
3. Run `npm run lint` to check for any issues
4. Run `npm run build` to ensure the build passes
5. Submit a pull request

## Code Style

This project uses ESLint, Prettier, and SwiftLint for code formatting. Please ensure your code follows the project's style guidelines before submitting a pull request.
